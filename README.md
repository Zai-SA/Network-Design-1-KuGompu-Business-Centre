# Network-Design-1-KuGompu-Business-Centre
This project is a simulation of a small enterprise newtork I named KuGompu Buisness centre. There are 5 small offices (accounting practice, law practice, coffee shop , call centre , and IT company) and each of  them are represented by VLANs. 

# Vlan:
10 - Accounting 
20 - Law
30 - Coffee shop
40 - Call Centre
50  - IT

# Key configurations
Inter-vlan-routing,
Dhcp configuration,
Vlan segmentation, 
Voice VLAN, 
Port security, 
Wireless Connections
